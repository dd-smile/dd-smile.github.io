Calendar日历
//创建一个Calendar的对象
Calendar cal = Calendar.getInstance(); //必须通过其静态方法创建,默认的,cal为当前时间

//获取年月日时分秒的值
//年
int year = cal.get(Calendar.YEAR);
//月:从0开始计算,范围0,1,2,...,11
int month = cal.get(Calendar.MONTH);
//天:本月第几天,范围1,2,3,...,31
int dayOfMonth = cal.get(Calendar.DAY_OF_MONTH);
//小时:24小时制,范围0,1,2,...,23
int hour = cal.get(Calendar.HOUR_OF_DAY);
//分钟:范围0,1,2,...,59
int minute = cal.get(Calendar.MINUTE);
//秒:范围0,1,2,...,59
int second = cal.get(Calendar.SECOND);
//星期几:本周第几天,范围1,2,...,7,星期日为1
int dayOfWeek = cal.get(Calendar.DAY_OF_WEEK);
//DAY_OF_YEAR:当年第几天,范围1,2,...,366
int dayOfYear = cal.get(Calendar.DAY_OF_YEAR);

//设置年月日时分秒
//设置年月日"2020-6-7",注意月份从0开始计数
cal.set(2020,5,7);
//设置年月日时分秒"2020-6-7 20:15:48"
cal.set(2020,5,7,20,15,48);
//只设置年
cal.set(Calendar.YEAR,2020);
//只设置月,注意月份从0计算
cal.set(Calendar.MONTH),5);
//只设置日
cal.set(Calendar.DAY_OF_MONTH,7);
//只设置时
cal.set(Calendar.HOUR_OF_DAY,20);
//只设置分
cal.set(Calendar.MINUTE,15);
//只设置秒
cal.set(Calendar.SECOND,48);

//往前或往后推算,也可以按小时,月等推算
//前天
cal.add(Calendar.DAY_OF_MONTH,-2);
//30天后
cal.add(Calendar.DAY_OF_MONTH,30);

捕获异常
try
{
    //要监视的代码块
}
catch(Exception e)    //这个参数表示可能出错的异常
{
    //若捕获到了异常,在此处理
}



//异常机制
1.立即从try{...}退出,类似break效果
2.进入catch(Exception e){...}
-Exception e,//表示捕获到的异常对象
-e.getClass().getName(),//异常类名
-e.getMessage(),//异常描述
-e.printStackTrace(),//打印函数栈找异常溯源
3.在try{...}catch{...}捕获处理之后,程序继续运行

按类型捕获

try{
    //监控代码
}
catch(ArithmeticException e)

}
catch(ArravIndexOutOfBoundsException e)
{
    //越界异常
}
catch(NullPointerException e)
{
    //空指针异常
}
//最好可以添加一个默认Exception来防止漏捕
catch(Exception e)
{
    //默认匹配,防止逃逸
}

finally

finally,与try{…}联用,执行一些异常清理工作

一般形式为:

try
{
        //要监视的代码块
}
catch(Exception e)
{
     //若捕获到了异常,在此处理
}
finally{
    //一些清理工作
}

其中,
//1.若try{...}中有异常发生
//则跳出try{}→执行catch{}→执行finally{}
//2.若try{...}中没有异常发生
//由执行try{}→执行finally{}

显然,finally{...}总是会执行,通常用于执行-清理善后-工作

文件与目录

File类是一个轻量级的类,不占资源.

file仅表示一个路径，并不会真的创建一个文件

显然,创建一个File对象不会包含文件中的数据,和文件没啥关系

​ java.io.File,用于描述一个文件、目录的路径

​ eg：

File file = new File("D:/example/123.txt");

File dir = new File("D:/picture/")
文件操作

-file.exists()，判断它是否存在

-file.isFile()/isDirectory()，判断它是文件、还是目录

-file.getName()，返回文件名，含后缀

-file.length()，返回文件的长度，单位是字节

-file.lastModified()，返回文件的修改时间，时间值（ms）

-file.getParentFile()，返回上级目录

目录操作

-dir.exists()，判断它是否存在

-dir.getName()，返回目录名

-dir.getParentFile()，获取上级目录

-dir.mkdir()，创建单个目录

-dir.mkdirs()，创建多层目录

​ 常使用这个

-dir.listFiles()，列出子文件/子目录

​ File[] files = dir.listFiles();

​ 有可能返回为null，最好要判断一下

{

    byte [] buffer = new byte[1200];
    File file = new File("sample.txt");
    
    FileInputStream fis = new FileInputStream(file);
    int n = fis.read(buffer);
    fis.close();
    
    //2.解码成文本
    String text = new String(buffer,0,n,"UTF-8");   //指示起始位置0,和长度n(n个字节),字符编码"UTF-8"
